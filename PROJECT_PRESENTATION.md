## 📋 **项目简介**

### **一句话介绍**

**AI Journal** - 像聊天一样写日记，让情绪记录变得简单自然。

### **解决的问题**

| 痛点 | 传统日记 | AI Journal |
|------|---------|-----------|
| **坚持难** | 需要主动构思，容易放弃 | AI 引导提问，降低门槛 |
| **情绪洞察** | 自己难以客观分析 | AI 实时情绪识别与反馈 |
| **隐私顾虑** | 担心被人看到 | 完全私密，AI 保密 |
| **互动缺失** | 单向记录，无反馈 | 双向对话，温暖陪伴 |
| **语言障碍** | 单一语言 | 中英文无缝切换 |

### **目标用户**

- 🧑‍💻 **职场人士** - 工作压力大，需要情绪出口
- 🎓 **学生群体** - 成长困惑，需要引导反思
- 🌍 **双语用户** - 中英文切换，跨文化表达
- 💕 **心理关注** - 重视心理健康，主动记录情绪

## 🤖 **AI 创新性**

### **1. 智能模型路由**

```
用户输入 → 日记类型识别 → 自动选择最佳模型
   ↓
📝 日常对话 → Claude Opus 4.6（最强推理）
💼 工作学习 → Claude Sonnet 4.6（平衡性能）
💕 情感倾诉 → Claude Opus 4.6（深度理解）
📚 学习成长 → GPT-4o（知识丰富）
✍️ 创意写作 → Claude Opus 4.6（创意写作）
```

**创新点**: 用户无需了解模型差异，系统自动选择最优解。

### **2. 动态 System Prompt**

根据场景自动调整 AI 角色：

| 类型 | AI 角色 | 回复风格 | 字数限制 |
|------|--------|---------|---------|
| 日常对话 | 温暖朋友 | 亲切、支持 | 100 字 |
| 工作学习 | 职业顾问 | 专业、条理 | 150 字 |
| 情感倾诉 | 心理倾听 | 共情、理解 | 120 字 |
| 学习成长 | 学习导师 | 清晰、启发 | 150 字 |
| 创意写作 | 创意伙伴 | 想象、开放 | 200 字 |

### **3. 多语言自适应**

```
中文用户 → Claude Opus 4.6（中文支持优秀）
英文用户 → GPT-4o（英文表达自然）
无缝切换，无需重新配置
```

**技术实现**:
```typescript
const getModel = (language: string, category: string) => {
  if (language === 'en') return 'openai/gpt-4o';
  return MODELS.byCategory[category];
};
```

### **4. 47 个模型池**

- 接入 GMI Cloud 47 个顶级模型
- 按需选择，成本优化
- 故障自动切换，高可用

**模型分类**:
- Claude 系列 (7 个) - 推理最强
- GPT 系列 (13 个) - 通用最佳
- Qwen 系列 (9 个) - 中文优化
- DeepSeek (3 个) - 性价比
- 其他 (15 个) - 特色场景

## 🛠️ **技术实现**

### **整体架构**

```
┌─────────────────────────────────────────────────────────┐
│                     用户浏览器                           │

                     │ HTTPS

│              Cloudflare Workers (后端路由)               │

│  - 模型智能路由                                         │
│  - API 认证 + 速率限制                                   │
│  - 全球 CDN 加速                                         │
└────────────────────┬────────────────────────────────────┘
                     │
                     ↓

│  - 47 个 LLM 模型（Claude/GPT/Qwen/DeepSeek...）           │
```

### **技术栈选型**

| 层级 | 技术 | 选型理由 |
|------|------|---------|
| **前端** | Next.js 14 | 静态导出，SEO 友好，快速加载 |
| **样式** | TailwindCSS | 原子化 CSS，快速开发 |
| **后端** | Cloudflare Workers | 全球边缘计算，低成本 |
| **框架** | Hono | 轻量快速，Workers 原生支持 |
| **部署** | Cloudflare Pages | 自动 CI/CD，免费额度高 |
| **模型** | GMI Cloud | 47 个模型，统一 API |

## 🎯 **Q&A 准备**

### **常见问题**

**Q1: 数据隐私如何保护？**

A: 我们采用纯前端 + Serverless 架构：
- 对话数据不落地，无数据库存储
- 所有通信 HTTPS 加密
- Cloudflare 全球 CDN，符合 GDPR
- 用户可随时清除浏览器数据

**Q2: 成本如何控制？**

A: 三层成本优化：
1. Cloudflare 免费额度（10 万请求/天）
2. GMI 按量付费（$0.0001/1K tokens）
3. 智能模型路由（简单任务用便宜模型）

单用户日均成本：< $0.01

**Q3: 如何盈利？**

A: 三层变现模式：
- **免费层**: 基础对话，每日 10 条
- **付费层**: $9.99/月，无限对话 + 高级模型
- **企业层**: $99/月，API 接入 + 定制模型

**Q4: 与竞品差异？**

A: 三大核心优势：
1. **智能模型路由** - 自动选择最优模型
2. **多语言原生支持** - 中英文无缝切换
3. **零运维成本** - Serverless 架构

**Q5: 技术壁垒是什么？**

A: 
- 模型路由算法（已实现）
- 用户行为数据积累（待积累）
- 情绪分析模型（可扩展）
